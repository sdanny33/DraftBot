# DraftBot

## Author

Daniel Soares. Email me if you have questions ([dcs3personal@gmail.com](mailto:dcs3personal@gmail.com)).

## Description

A Discord bot to leave and pick for Pokémon Draft Leagues. The Discord bot is connected to a Google Sheet and when a player runs a command it updates the sheet accordingly.

## How to use

Add it to your discord server, setup the Google sheet you would like the output to go to, and run some commands!
